# TPC-01 Firmware Updates

Private release repository for signed TPC-01 M5Dial firmware updates.

Each GitHub Release must contain both files:

- `tyre_pressure_idf.bin`
- `firmware-manifest.json`

The manifest records the firmware version, exact byte count, and SHA-256 hash. It is signed with the dedicated ECDSA P-256 private key held in the private source repository's GitHub Actions secrets. M5Dial firmware contains only the matching public key and rejects missing, modified, unsigned, or incorrectly signed files.

Releases are generated by `.github/workflows/publish-m5dial-firmware.yml` in the private `Tyre-Pressure-Control` source repository. Do not manually alter release assets after publication.

Keep this repository private. Never commit GitHub access tokens or the firmware signing private key. Device/customer tokens must be fine-grained, restricted to this repository, and set to `Contents: Read-only`.
